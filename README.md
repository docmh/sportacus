# sportacus
Den Prompt zum custom GPT "Sportacus" findest du in der Datei `sportacus_prompt.md`. 

Personalisiere diesen Prompt und erstelle einen Custom GPT auf ChatGPT oder ein Gem auf Gemini. 

Aktiviere unbedingt die Canvas-Option, damit das erstellte Programm auch in deiner Chatbot-UI ausgeführt werden kann. Mit aktivierter Websuche kann Sportacus auch im Internet nach Evidenz für das Training suchen. 

Verwende ein Reasoning-Modell mit hohem Reasoning-Effort für beste Ergebnisse. 

Achtung: Canvas Apps funktionieren in ChatGPT nur in der Desktop/Web-Oberfläche, noch nicht mobil. 

Hinweis: Selbstverständlich ist dies kein medizinisches Angebot, Verwendung geschieht auf eigene Verantwortung; im Zweifelsfalle: kläre den Gebrauch zuvor mit der Ärztin/dem Arzt deines Vertrauens ab.
